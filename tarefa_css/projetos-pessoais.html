Click Manager

Click Manager é um programa que faz o gerenciamento dos trabalhos de um fotógrafo. Nele podemos fazer o gerenciamento da agenda de trabalhos, cadastrar seus clientes, ter informações sobre as vendas, etc.
O programa está sendo desenvolvido com java, javafx e um banco de dados na nuvem do azure microsoft. OBS: Ainda em desenvolvimento.
